Ejercicios Javascript - Oposiciones

Ejercicio 5 - Oposición 2008

Realitze una pàgina web utilitzant javascript que permet a l'usuari encertar un número situat entre un valor máxim i un valor mínim (assignats al codi). Si el número introduït està for adel rang dels valors màxim i mínim se li tornarà a demanar un número, sense que això compte com a intent. Es donarà un màxim de 3 intents.

El missatge final a mostrar haurà de ser un d'aquests dos missatges:

  • "Enhorabona, has encertat en l'intent X"
  • "Ho sent, has fallat després de 3 intents"
